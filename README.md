# ✋ Hexis — Hand Gesture Control System

Hexis is a real-time **computer vision-based interaction system** that allows users to control a computer using hand gestures captured through a webcam. It replaces traditional input devices (mouse, keyboard) with intuitive, AI-powered hand tracking.

The system is modular and supports multiple “ability modes” such as cursor control, media control, and experimental interaction systems.

---

## 🚀 Features

- ✋ Real-time hand tracking using webcam
- 🧠 Gesture recognition using computer vision logic
- 🖱️ Cursor control using hand movement
- 👆 Gesture-based click and interaction system
- 🎛️ Modular ability system (expandable controls)
- 🎨 Visual overlay system for feedback
- ⚡ Lightweight and runs in real-time

---

## 🧠 How It Works

1. Webcam captures live video feed
2. Hand is detected and tracked frame-by-frame
3. Key landmarks of the hand are extracted
4. Gesture logic interprets finger positions and movement
5. Actions are mapped to system controls (mouse, effects, etc.)

---

## 🛠️ Tech Stack

- Python 🐍
- OpenCV (video processing)
- MediaPipe (hand tracking model)
- NumPy (mathematical processing)
- PyAutoGUI (system control automation)
